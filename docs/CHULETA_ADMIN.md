# CHULETA DE CAJA — corregir algo ya hecho

> Para tener detrás del mostrador. Si te equivocaste en algo, aquí está dónde se arregla.
> Casi todo lo de abajo es solo para **administradores**. Si no ves el botón, es por permisos.

---

## CORREGIR ALGO YA HECHO

| Lo que pasó | Dónde se arregla |
|---|---|
| **Cliente mal en un ticket** | Tickets → fila → **Editar** (lápiz) → cambia cliente → *Guardar cambios* |
| **Método de pago mal** (puse tarjeta y fue efectivo) | Tickets → fila → **Pagos** → corrige método/importe (tiene que poner *"Cuadra"*) → *Guardar pagos* |
| **Producto / cantidad / precio mal** | Tickets → fila → **Líneas** → corrige → escribe el nº de ticket → *Guardar líneas* |
| **Anular un ticket entero** | Tickets → fila → **Eliminar** (rojo) → lee el resumen → escribe el nº de ticket → *Eliminar definitivamente* |
| **Editar / borrar una retirada de caja** | Contabilidad → **Resúmenes de Caja** → abre la sesión → en Movimientos, **lápiz** / **papelera** de la retirada |
| **Corregir el arqueo / registrar un descuadre real** | Contabilidad → **Resúmenes de Caja** → abre la sesión → **Editar arqueo** → pon el efectivo realmente contado → *Guardar* |
| **Reabrir una caja cerrada por error** | Contabilidad → **Resúmenes de Caja** → abre la sesión → **Reabrir** (candado) |
| **Editar / borrar un cobro de un pedido** | Pedidos → abre el pedido → pestaña **Pagos** → *Historial de pagos* → **lápiz** / **papelera** |
| **Saldo de un vale mal** | Vales → abre el vale → **Ajustar saldo** → nuevo saldo + motivo (mín. 10 caracteres) → *Aplicar ajuste* |
| **Reactivar / anular un vale** | Vales → abre el vale → **Reactivar** (si estaba anulado) o **Anular** |
| **Dos fichas del mismo cliente** | Clientes → abre la que **sobra** → **Fusionar con…** → elige la que se queda → escribe el nombre → *Fusionar definitivamente* ⚠️ |
| **Ajuste de stock equivocado** | Stock → pestaña **Movimientos** → fila del ajuste → **Revertir** |
| **Registrar un gasto / asiento manual** | Contabilidad → pestaña **Asientos** → **Nuevo asiento** → líneas que cuadren (Debe = Haber) → *Crear asiento* |

---

## TRES REGLAS DE ORO

1. **Caja descuadrada:** nunca falsees el conteo para que cuadre. Cierra normal y **corrige el arqueo desde Contabilidad** poniendo el efectivo real. Así queda constancia.

2. **Fusionar clientes NO se puede deshacer.** Es lo único irreversible de todo el sistema. **Confírmalo con otra persona** antes de pulsar *Fusionar definitivamente*.

3. **Si el sistema te frena, lee el mensaje en rojo.** Casi siempre te dice exactamente qué pasa (factura en Hacienda, stock que ya se movió, caja ya abierta…).
